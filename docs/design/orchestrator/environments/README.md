# Environment Design

This directory contains design decisions and rationale for environment implementations.

## Contents

- [bash.md](./bash.md) - Bash environment design (shell command execution)
- [python.md](./python.md) - Python environment design (REPL for data analysis)
- [editor.md](./editor.md) - Editor environment design (query-based file viewing/editing)
- [system.md](./system.md) - System environment design (project context display)

## Purpose

Environment design documentation explains **WHY** decisions were made about:
- Architecture decisions for specific environments
- Design rationale for environment behavior
- Trade-offs in environment implementation
- Cross-cutting concerns affecting environment design

## What Doesn't Go Here

- Implementation details (belongs in source code docstrings)
- Task definitions (belongs in [tasks/](../../../tasks/))
- User guides (belongs in [guides/](../../../guides/))
